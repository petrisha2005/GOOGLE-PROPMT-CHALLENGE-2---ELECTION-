# 🗳️ CivicPulse

LIVE DEMO:
https://civic-pulse-84b38.web.app


## 🚀 Overview
CivicPulse is an interactive, visual-first voting rehearsal platform designed to make the democratic process transparent, accessible, and engaging for all electors.

The platform combines gamified tracking paths, hardware-accurate visual sandboxes, and premium multi-lingual audio guidance to bridge the gap for first-time voters, low-literacy individuals, and citizens navigating new electoral systems globally.

---

## 🎯 Problem Statement
Many individuals struggle to:

*   Understand complex, text-heavy election regulations
*   Track state-by-state or country-specific voting timelines
*   Identify required identity verification documents
*   Build the confidence and physical muscle memory to vote
*   Navigate technical digital voting systems or physical machines

CivicPulse simplifies this process through highly visual, interactive learning pathways and risk-free simulation environments.

---

## ✨ Key Features

### 📊 Interactive Visual Tracker
*   Winding board-game style roadmap
*   Zero-text visual pipelines
*   Tactile universal pictograms
*   Dynamic, illuminating progress tracks

### 🎙️ Premium Audio Guide
*   "Press & Listen" omnipresent speaker icons
*   Natural human voice tones (bypasses robotic TTS)
*   Slower, patient conversational pacing
*   Multi-lingual and localized accent support

### 🌍 Global Sandbox Modules
*   **🇮🇳 India (Visual EVM Track):** Realistic Electronic Voting Machine mockup featuring recognizable party symbols, tactile blue voting buttons, red flashing LEDs, and authentic confirmation beeps.
*   **🇺🇸 United States (Mail-In Track):** Interactive envelope tracker paired with an HTML5 canvas touch/mouse drawing pad for practicing legal affidavit signature verification.
*   **🇪🇪 Estonia (Digital i-Voting Track):** A futuristic online client simulation requiring users to enter security codes to practice cryptographic PIN 1 (authentication) and PIN 2 (signing) workflows.
*   **🇦🇺 Australia (Ranked-Choice Track):** A preferential voting sandbox where users drag, drop, and rank candidates numerically ($1^{\text{st}}, 2^{\text{nd}}, 3^{\text{rd}}$) to learn vote redistribution mechanics.

### 🔒 Local-First Privacy Protection
*   Strict Zero-Log Policy
*   No database storage for personal signatures or selections
*   Runs exclusively in temporary browser memory
*   100% risk-free rehearsal workspace

## 📌 Workflow

1. User selects their global country voting track on the dashboard
2. CivicPulse launches the winding visual journey roadmap
3. Conversational companion reads instructions aloud via the audio icon
4. User steps through localized registration checks and booth locators
5. Dynamic country-specific gameplay sandbox is loaded (EVM, Envelope, or Keypad)
6. User practices the physical act of voting (pressing buttons, signing, or sorting)
7. Success indicators (beeps, LEDs, or encryption animations) confirm the action
8. User completes the journey by unlocking the counting and results station
